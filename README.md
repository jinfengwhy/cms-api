# CMS-API

## 项目介绍

* 一个简单论坛系统的后台接口开发
* 类似于 https://cnodejs.org/
* 实现注册、登录、话题资源管理、评论资源管理

## 项目启动

1. 本地启动 mysql 数据库
2. 运行项目根目录下的 cms.sql 文件，创建数据表
3. 执行命令 `node app.js` 启动 API 服务
4. 访问 `http://127.0.0.1:3000/topics` 测试 API 服务是否可用
